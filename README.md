# 💰 Expense Tracker

A full-stack expense management web application built with React, Node.js, Express and MongoDB.

## 🛠️ Tech Stack

- **Frontend:** React, Vite, Recharts
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose)

## ✨ Features

- Add, Edit, Delete expenses
- Filter by category and date range
- Monthly summary (Total, Highest, Top Category)
- Bar chart visualization (Recharts)
- Budget alerts when limit exceeded
- CSV export
- Responsive dark UI

## 📁 Project Structure
expense-tracker/
├── client/         → React Frontend
└── server/         → Node.js Backend
## 🚀 Setup & Run

### Backend
```bash
cd server
npm install
node server.js
```

### Frontend
```bash
cd client
npm install
npm run dev
```
